CDR-ANALYTICS-DASHBOARD

A SaaS-style dashboard for analyzing telecom Call Data Records (CDR).

## Features
- KPI Summary Cards
- Call Duration Analytics
- Call Cost by City
- Call Activity Timeline
- Calls by City Pie Chart
- Recent Call Logs Table

## Tech Stack
- React + Vite
- Tailwind CSS
- shadcn/ui
- Recharts

## Deployment
Hosted on Vercel.

## Run Locally
```bash

npm install
npm run dev
